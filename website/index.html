Three ideas, one surface.

① Cells run commands

r: cmd runs once. i: cmd streams forever — live subprocess output, capped at 200 lines. The output is the cell value. Diff it, copy it, export it to CSV. No panel, no terminal pane, no context switch.

② Always visible, never in the way

Lives at the bottom of the Z-order — behind every other window, on top of the wallpaper. Win+D doesn't dismiss it. Alt+Tab skips it. On Linux it sets _NET_WM_WINDOW_TYPE_DESKTOP. It's just there, like a sticky note that does math.

③ Plain CSV, all the way down

No database, no JSON state, no cloud. Your sheet is one CSV file you can git diff. Autosaves every 5s in desktop mode. Survives reboots. Sync with rsync, Dropbox, or nothing at all.

The prefix language.

r:Run a command once. Output replaces the cell.
i:Inline live process. Stdout streams into the cell, capped at 200 lines. ConPTY on Windows, pipe redirect on Linux.
http(s)://Auto-detected hyperlink. Open it from the cell.
{A1::C10}Cell-range reference. Splice grid contents into any command.
Σ / ΠColumn sums and row products, computed live.

Install.

Build from source today. .NET 9 SDK required. Pre-built binaries are coming once code-signing is in place.

Windows

git clone https://github.com/cemheren/QuickSheet
cd QuickSheet
dotnet run -c Release --project ExcelConsole.csproj -- --desktop

Embeds into the WorkerW behind the wallpaper.

Linux

git clone https://github.com/cemheren/QuickSheet
cd QuickSheet
dotnet run -c Release --project ExcelConsole.csproj -- --desktop

X11 only. Wayland support is on the roadmap.

Console TUI

dotnet run --project ExcelConsole.csproj
# or with a CSV
dotnet run --project ExcelConsole.csproj -- data.csv

Same grid, plain terminal. Works anywhere .NET 9 runs.

Why you can trust the binary.

  • Zero NuGet dependencies. Every line is in the repo. Native interop is hand-written P/Invoke (libX11, Win32).
  • Local-first. No telemetry. No accounts. No sync server. No network calls at all in the core.
  • CSV is the only state. No hidden database, no opaque format. Open it in any editor.
  • MIT-licensed. Read it, fork it, ship it. The repo is the whole product.
